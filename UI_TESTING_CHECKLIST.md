## Known Issues Found During Audit

| # | File | Line | Severity | Issue |
|---|---|---|---|---|
| 1 | `CollectAmountModal.vue` | 204 | **High** | `t('payments.methods.${m.value}')` references missing i18n key `payments.methods.*` — not present in `en.json`. Renders raw key string. |
| 2 | `ModuleDetailModal.vue` | 78 | Medium | Modal `content: 'sm:max-w-3xl'` missing `max-h-[80vh]` and `overflow-y-auto`. Operation log can overflow on short screens. |
| 3 | `ModuleConfirmModal.vue` | 95 | Medium | Modal has no `ui` body scrolling config. Long dependency lists may extend past viewport. |
| 4 | `TreatmentEditModal.vue` | 117 | Low | Modal missing explicit `max-h` and `overflow-y-auto`. Notes and surface buttons could overflow on small viewports. |
| 5 | `default.vue` (USlideover) | 191 | Low | Mobile nav slideover missing `overflow-y-auto` on the outer flex container. The `nav` has `overflow-y-auto` but the full-height flex column could still overflow. |
| 6 | `SearchBar.vue` | 29 | Low | Uses raw `let` timer variable instead of VueUse `useTimeoutFn`. No memory leak (cleanup is present) but inconsistent with potential composable pattern. |

## Testing Configuration Notes

- **Viewport sizes**: Desktop = 1280×800, Tablet = 768×1024, Mobile = 375×812
- **Auth fixture**: Use `test.use({ role: 'admin' })` for admin-gated pages
- **Locale**: Default locale is `en` (`i18n.strategy: 'no_prefix'`)
- **API base**: `http://localhost:8000` (backend)
- **E2E runner**: Playwright via `npm run test:e2e`
- **Demo credentials**: `admin@demo.clinic` / `demo1234`
